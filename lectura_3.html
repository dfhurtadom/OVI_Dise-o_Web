SUBLIME TEXT


Sublime Text es un editor de código multiplataforma, este editor nos permite tener varios documentos abiertos mediante pestañas,podemos utilizarlo en pantalla completa y aprovechar todo el espacio visual de la pantalla.


El sistema resaltado de sintaxis de Sublime Text soporta un gran número de lenguajes (C, C++, C#, CSS, D, Erlang, HTML, Groovy, Haskell, HTML, Java, JavaScript, LaTeX, Lisp, Lua, Markdown, Matlab, OCaml, Perl, PHP, Python, R, Ruby, SQL, TCL, Textile and XML)

Algunas de sus caracteristicas mas destacadas son:

*Autocompletado:Esta funcion nos ayuda a recordar como se escriben ciertos comandos,funciones o textos del lenguaje en el que nos encontramos.
*Selecciones múltiples:Una característica muy interesante para realizar tareas de una forma sencilla y rápida,si seleccionamos una palabra con el mouse, pulsando CTRL+D podremos ir seleccionando las siguientes coincidencias y modificarlas todas a la vez.
*Auto-cerrado de etiquetas:la combinación de teclas ALT+. puede ayudarnos a cerrar rápidamente cualquier etiqueta HTML que tengamos abierta de forma instantánea.
*Code-folding:Sublime Text también proporciona la posibilidad de plegar o desplegar bloques de código, tanto en HTML como en lenguajes de programación.
